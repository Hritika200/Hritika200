## Projects

### 🔍 [Browser Extension Security — Automated SAST/DAST Scanner](https://github.com/Hritika200/Browser-Extension-Security)

Python framework that scans JavaScript files across Chrome extensions for unsafe permission patterns, privilege escalation, and sensitive API misuse. AST-based static analysis mapped to OWASP and CWE categories, plus a dynamic validation layer to reduce false positives.

- **2,000+ JS files** scanned across **30+ extensions**
- Hybrid static-dynamic pipeline with risk-ranked output reports

---

### 🛡️ [Zero-Day Attack Detection — Stacking Ensemble IDS](https://github.com/Hritika200/Zero-Day-Attack-Detection)

Meta-learning stacking ensemble (Extra Trees + Random Forest + AdaBoost → Logistic Regression) trained on the X-IIoTID dataset to detect previously unseen attack categories. SHAP explainability layer makes model decisions auditable.

- **96%+ accuracy** on 3 unseen zero-day attack groups
- **0.96–0.964 F1** across unknown threat patterns

---

### ☁️ [Secure DevOps — Cloud Infrastructure + Observability](https://github.com/Hritika200/Secure_DevOps)

GCP VM provisioned with Terraform, containerised full-stack app, ELK Stack + Fluent Bit for log indexing across frontend and backend, New Relic APM for backend monitoring. End-to-end IaC and observability from commit to dashboard.
