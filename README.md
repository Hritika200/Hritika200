## Projects worth looking at

### 🔍 [Browser Extension Security — Automated SAST/DAST Scanner](https://github.com/Hritika200/Browser-Extension-Security)
Built a Python framework that scans JavaScript files across Chrome extensions for unsafe permission patterns, privilege escalation, and sensitive API misuse. Uses AST-based static analysis mapped to OWASP and CWE categories, plus a dynamic validation layer to cut false positives.
- **2,000+ JS files** scanned across **30+ extensions**
- Hybrid static-dynamic pipeline with risk-ranked output reports

---

### 🛡️ [Zero-Day Attack Detection — Stacking Ensemble IDS](https://github.com/Hritika200/Zero-Day-Attack-Detection)
Trained a meta-learning stacking ensemble (Extra Trees + Random Forest + AdaBoost → Logistic Regression) on the X-IIoTID dataset to detect previously unseen attack categories. Added SHAP explainability to make the model's decisions actually auditable — not just accurate.
- **96%+ accuracy** on 3 unseen zero-day attack groups
- **0.96–0.964 F1** — strong generalisation to unknown threat patterns

---

### ☁️ [Secure DevOps — Cloud Infrastructure + Observability](https://github.com/Hritika200/Secure_DevOps)
Provisioned a GCP VM with Terraform, containerised a full-stack app, and wired up ELK Stack + Fluent Bit to index logs from both frontend and backend. Integrated New Relic APM for backend monitoring. End-to-end IaC + observability, from commit to dashboard.
